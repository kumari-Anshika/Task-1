# KofiShop Landing Page ☕

## Overview

KofiShop Landing Page is a modern and responsive coffee shop website designed using HTML5 and CSS3. The project showcases a visually appealing landing page for a coffee shop, featuring a hero section, navigation menu, promotional content, and social media links.

The website is fully responsive and adapts seamlessly to desktops, tablets, and mobile devices.

## Features

* Responsive navigation bar
* Attractive hero section with call-to-action button
* Modern coffee shop themed UI
* Promotional cards and featured content
* Social media integration
* Fully responsive design using Flexbox and Media Queries
* Clean and user-friendly interface

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries
* Font Awesome

## How to Run the Project

1. Clone or download the repository.
2. Open the project folder in Visual Studio Code.
3. Install the Live Server extension.
4. Right-click on `index.html`.
5. Select "Open with Live Server".
6. Resize the browser window to test the responsiveness.


## Responsive Design

The webpage is optimized for:

* Desktop Devices
* Tablets
* Mobile Phones

Media Queries are used to ensure a smooth user experience across different screen sizes.
